# Nexara – Gamified Data Science Learning Platform

## Overview
Nexara is a gamified learning platform designed to help students learn data science concepts through structured learning paths, challenges, and XP-based progression.

The platform motivates learners by introducing game mechanics like levels, experience points, and leaderboards.

## Features

- User Authentication (Email + Google Login)
- JWT-based secure authentication
- Gamified dashboard with XP and levels
- Learning paths for Data Science topics
- Interactive modules with videos and articles
- Challenge system with XP rewards
- Leaderboard for ranking users

## Tech Stack
